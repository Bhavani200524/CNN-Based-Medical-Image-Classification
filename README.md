# CNN-Based-Medical-Image-Classification
Deep learning model for classifying brain tumors (glioma, meningioma, pituitary, no tumor) using MRI data.

##  Overview
This project presents a deep learning-based approach for automatic brain tumor classification using MRI images.  
A Convolutional Neural Network (CNN) model is implemented in MATLAB to classify brain tumors into four categories: glioma, meningioma, pituitary tumor, and no tumor.

The system reduces manual effort and improves diagnostic accuracy using AI techniques.

##  Objectives
- Automate brain tumor detection using deep learning  
- Improve diagnostic accuracy using CNN  
- Reduce manual analysis time  
- Classify MRI images into multiple tumor categories  

##  Key Features
- CNN-based image classification  
- MRI image preprocessing (resizing, normalization)  
- Multi-class classification (4 classes)  
- High accuracy (~92.29%)  
- MATLAB implementation  

##  Dataset
- Source: Kaggle Brain Tumor MRI Dataset  
- Classes:
  - Glioma  
  - Meningioma  
  - Pituitary Tumor  
  - No Tumor  

- Data Split:
  - Training: 80%  
  - Testing: 20%  

## Methodology

### Step 1: Image Preprocessing
- Resizing images  
- Normalization  
- Noise removal  

###  Step 2: CNN Model
- Convolution Layers  
- ReLU Activation  
- Pooling Layers  
- Fully Connected Layer  
- Softmax Classifier  


### Step 3: Classification
- Model predicts tumor type based on MRI image  
- Outputs probability scores for each class  


##  Results

- Improved classification performance
